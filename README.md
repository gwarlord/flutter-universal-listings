# instaflutter
Instaflutter project.

## Setup Flutter

1. Install Flutter SDK from [Flutter's official website](https://docs.flutter.dev/get-started)
2. Add Flutter to your system PATH
3. Run `flutter doctor` to verify installation and dependencies
4. Set up your preferred IDE (Android Studio/VS Code) with Flutter plugins

## Run the Project

1. Run `flutter pub get` to install dependencies
2. Connect a device or start an emulator
3. Run `flutter run` to start the app
